# Cricket Data Scraping and IPL Performance Analysis

## Overview

* Analysis of Indian Premier League (IPL) team performance for the 2022–2024 seasons.
* Focus on identifying sustained team dominance and consistency.

## Data Source

* Official IPL points table data.

## Method

* Data collected using a web scraping extension and organized in Excel.
* Data cleaning and preprocessing performed using Excel functions.
* Performance metrics and indices calculated to compare teams across seasons.

## Key Metrics

* Win Percentage
* Net Run Rate (NRR)
* Batting Strength
* Bowling Strength
* Balanced Performance Index
* Season Dominance Index (SDI)

## Key Insight

* **Rajasthan Royals** and **Kolkata Knight Riders** show strong and balanced performance across seasons.

## Tools Used

* Microsoft Excel
* Data Visualization
